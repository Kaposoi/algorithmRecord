# ajax的运行原理

一般的交互

浏览器 ->(请求) 服务器

服务器 ->(响应) 浏览器

开发人员不可控



基于ajax的交互

浏览器端 ->(创建) ajax ->(请求) 服务器端

服务器端 ->(响应) ajax ->(响应) 浏览器端



ajax的实现步骤

>创建ajax对象
>
>```
>var xhr = new XMLHttpRequest()
>```
>
>告诉ajax请求地址和请求方式
>
>```
>xhr.open('get', 'http://www.example.com')
>```
>
>发送请求
>
>```
>xhr.send()  // 如果不调请求是不会发送的
>```
>
>获取服务器端和客户端的响应数据
>
>```
>xhr.onload = () => {
>	...
>}
>```
>
>注意，这是一个回调函数，获得响应后才执行这个函数
>
>xhr.responseText 用于接受服务器的响应文本。
