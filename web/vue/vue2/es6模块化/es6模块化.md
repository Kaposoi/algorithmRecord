# es6模块化

每个js文件都是模块， 用import导入， export导出

>es6没有require

## 默认导出和默认导入

```
export default 导出的成员
import 接收名称 from '模块标识符'
```

>每个模块中只能使用一次export default
>
>接收名称可以是任意名称， 只要符合标识符规定



## 按需导出和按需导入

```
export 按需导出的成员
import { s1 } from '模块标识符'
// s1和导出对应, 名字不可改变
```

>可以多次按需导出
>
>按需导入可以用as重命名
>
>import 中有{}是按需， 否则是默认



## 只执行代码

如果只想执行模块中的代码， 而不需要成员

```
import '模块标识符'
```
