# jquery选择器

li:first 获取第一个li

:last 获取最后一个li

:eq(index)  获取索引li

:odd 获取奇数

:even 获取偶数





.parent() 查找父级

children(selector) 最近一级

find(selector) 后代选择器

siblings(s) 查找兄弟结点， 不包括自己

nextAll 查找当前元素之后的所有的同辈元素

prevtAll 查找当前元素之前所有的同辈元素

hasClass("") 检查当前的元素是否含有某个特定的类

eq(index) 选取符合当前选择器的第index-1个选择器
